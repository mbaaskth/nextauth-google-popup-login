## Quick start

```bash
git clone https://github.com/arye321/nextauth-google-popup-login
```
rename `.env.example` to `.env` and put your google API things

to install project:

```bash
yarn
#or
npm install
```

to see the webpage:
```bash
yarn dev
#or
npm run dev
```
